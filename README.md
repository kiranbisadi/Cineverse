## 🎬 About the Project

**Cineverse** is a modern movie discovery web application built using **React**, **Vite**, **Tailwind CSS**, and the **TMDB API**. It enables users to explore popular movies, search for movies in real time, and view detailed information about each movie through a clean, responsive, and user-friendly interface.

The application demonstrates core React concepts such as component-based architecture, state management with `useState`, side effects using `useEffect`, client-side routing with React Router, API integration using Axios, and responsive UI development with Tailwind CSS.

### Key Features

* 🎥 Browse the latest popular movies
* 🔍 Search movies instantly
* 📄 View detailed movie information
* ⭐ Movie ratings and vote count
* 📅 Release date and runtime
* 🎭 Genres and production companies
* 💰 Budget and revenue details
* 🔙 Back navigation from the details page
* ❌ "No Movies Found" message for unsuccessful searches
* 🚫 Custom 404 Not Found page
* 📱 Responsive design for desktop, tablet, and mobile devices

### Technologies Used

* React.js
* Vite
* JavaScript (ES6+)
* Tailwind CSS
* React Router DOM
* Axios
* TMDB API
